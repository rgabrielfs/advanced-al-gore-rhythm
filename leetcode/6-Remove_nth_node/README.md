# Problema: Remove Nth Node from End of List

Autor: **João Izidoro**.

Revisado por:

Dada a `head` de uma lista encadeada, remova o `nº` do final da lista e retorna a `head`.

# Exemplo:

## Primeiro:

**Entrada:** head = [1,2,3,4,5], n = 2

**Saída:** [1,2,3,5]  

**Explicação:** Remove o 2o item da lista de trás pra frente, e retorna a lista sem esse valor.

## Segundo:

**Entrada:** head = [1], n = 1

**Saída:** []

# Restrições
* The number of nodes in the list is `sz`.
* `1 <= sz <= 30`
* `0 <= Node.val <= 100`
* `1 <= n <= sz`
